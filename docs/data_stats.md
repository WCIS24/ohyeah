# Data Statistics (FinDER)

Purpose:
- Provide citable dataset split sizes and subset statistics for the method chapter.

How to use:
- Cite `docs/data_stats.json` for total/split sizes and complex subset ratio; use the source paths for provenance.

## Source artifacts
- outputs/20260202_165254_c4b131/data_stats.json (generated by `scripts/prepare_data.py`)
- outputs/20260202_165303_b04a7b/subsets_stats.json (generated by `scripts/build_subsets.py`)

## Generation command
```powershell
python scripts\smoke.py --config configs\smoke.yaml
python scripts\prepare_data.py --config configs\prepare_data.yaml
python scripts\build_subsets.py --config configs\build_subsets.yaml
python scripts\compute_data_stats.py --config configs\data_stats.yaml
```

## Output schema
- docs/data_stats.json
  - source: {data_stats_path, subsets_stats_path, generated_by, git_hash}
  - totals: {total_queries, dev_total}
  - splits: {train, dev, test}
  - query_length: {count, min, max, mean, p50, p90}
  - evidence_count: {count, min, max, mean, p50, p90}
  - query_flags: {year_ratio, digit_ratio, percent_ratio}
  - dev_subsets:
    - complex: {size, ratio, rule_hits, path}
    - abbrev: {size, ratio, path}
