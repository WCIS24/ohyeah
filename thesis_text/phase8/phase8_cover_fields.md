# Phase 8 Cover Fields

Purpose:
- 记录封面可替换字段与默认值。

How to use:
- 将 cover.tex 中变量替换为真实信息即可，无需改版式。

| Field | Macro | Default | Description |
| --- | --- | --- | --- |
| Title | \thesistitle | TBD | 论文题目 |
| Department | \department | TBD | 院系 |
| Major | \major | TBD | 专业 |
| Class | \classname | TBD | 班级 |
| Student Name | \studentname | TBD | 学生姓名 |
| Student ID | \studentid | TBD | 学号 |
| Advisor | \advisorname | TBD | 指导教师 |
| Date | \dateSubmitted | \today | 提交日期 |
